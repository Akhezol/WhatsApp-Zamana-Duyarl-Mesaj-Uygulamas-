# WhatsApp-Zamana-Duyarl-Mesaj-Uygulamas-
Whatsapp Rehberinizdeki kişinin kaydettiğiniz gibi ismini, ona göndereceğiniz mesajı ve saati giriyorsunuz ardından o saatte uygulama bilgisayarınızdan whatsapp'ı açıp ona mesaj yazıyor. Uygulamanın sizden istediği çalıştırdığınızda bilgisayarınıza dokunmayın veya bir uygulama açmayın arkada açık olabilir ama bir eylem gerçekleştirmeyin!!
